About This Visualizer

This tool is designed to help you visualize and interact with tree data structures. You can build, edit, and manipulate Binary Trees and Binary Search Trees in real-time.

Features include node creation, editing, deletion, BST balancing, and Python code generation. It's built with HTML, Tailwind CSS, and vanilla JavaScript.
